# FigureLibraryTests
Этот маленькая библитека, которую можно использовать для вычисления площадей фигур. Этот проект был выполнен согласно заданию ниже:

Напишите на C# библиотеку для поставки внешним клиентам, которая умеет вычислять площадь круга по радиусу и треугольника по трем сторонам. Также проект должен в себе содержать:
- Юнит-тесты
- Легкость добавления других фигур
- Вычисление площади фигуры без знания типа фигуры в compile-time
- Проверку на то, является ли треугольник прямоугольным

### Использованные технологии и библиотеки
.NET 6.0, xUnit 2.4.2
## Запуск тестов
В папке с решением (FigureMindbox.sln) можно запустить команду:
```
    dotnet test
```
Или при использовании Visual Studio нужно перйти во вкладку "Тест" и выбрать пункт "Запуск всех тестов"
